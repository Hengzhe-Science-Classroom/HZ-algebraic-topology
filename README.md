## Features

- **Interactive Visualizations** — Path homotopy deformations, covering space lifts, boundary operator animations, homology calculations, cup product ring structure, Poincaré duality, fiber bundles
- **Rigorous Theory with Geometric Intuition** — Every abstract definition is grounded in visual examples before formalization
- **Computational Focus** — Learn to compute π₁, homology, and cohomology groups through interactive examples
- **Classic and Modern** — From fundamental group to spectral sequences and characteristic classes
- **Bilingual Presentation** — Chapter titles in Chinese and English; mathematical exposition accessible to both audiences
